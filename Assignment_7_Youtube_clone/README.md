# YouTube Clone

A **responsive YouTube-inspired web interface** built with **HTML, CSS, and modern design principles**. This project recreates the layout, navigation, and card-based video presentation of YouTube, with **animations and smooth transitions** for a modern user experience.

## Features

- Fully **responsive layout** for desktop, tablet, and mobile devices  
- **Sidebar navigation** with icons and labels  
- **Search bar** with focus animation  
- **Mic button** with hover effect  
- **Scrollable tags section** for video categories  
- **Video cards** with hover lift and shadow effects  
- **Avatar animations** for interactivity  
- Smooth transitions for all layout elements  

---

## Technologies Used

- **HTML5** – semantic page structure  
- **CSS3** – styling, animations, and responsiveness  
- **Google Fonts** – using `Roboto` font for modern typography  
- **Flexbox & CSS Grid** – responsive layouts  
- **CSS Transitions & Transformations** – hover effects and smooth animations

Responsive Design

The project uses CSS Grid and Flexbox for responsiveness:

Large screens (≥1200px): Fixed sidebar, full card layout

Tablet (≤992px): Adjusted search input, reduced card width

Mobile (≤768px): Bottom fixed sidebar, hidden text labels, stacked layout

Extra Small (≤480px): Smaller cards, responsive fonts

Animations & Interactivity

Cards: Lift up with shadow on hover (transform: translateY(-8px))

Avatar Images: Scale and rotate slightly on hover

Search & Mic Buttons: Scale effect on hover

Sidebar Icons: Smooth rotation and scale animation on hover

Tags: Smooth scale animation on hover

These animations improve user experience and create a dynamic, modern interface.

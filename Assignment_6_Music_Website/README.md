# myTunes Website

## Overview
**myTunes** is a responsive, modern web interface for showcasing music, movies, TV shows, and gift cards. Inspired by popular media platforms, it provides a clean and attractive layout, intuitive navigation, and engaging animations for an immersive user experience.

This project demonstrates:
- Responsive design for mobile, tablet, and desktop devices.
- Modern CSS techniques including Flexbox, Grid, and custom properties.
- Smooth animations and hover effects.
- Semantic HTML5 structure and accessibility considerations.

## Features
- **Responsive Layout:** Works seamlessly on phones, tablets, and desktops.
- **Sticky Navigation:** Easy navigation across sections.
- **Hero Section:** Full-screen background with animated heading and description.
- **Music Section:** Highlight music streaming services with a call-to-action button.
- **About Section:** Showcase movies and TV collections.
- **Video Section:** Display video content with visually appealing background.
- **Gift Cards Section:** Showcase gift cards with buttons for redemption.
- **Footer:** Multi-column footer with navigation links and contact info.
- **Animations:** Smooth fade-in, slide-up, and floating effects.

---

## Technologies Used
- **HTML5** – Semantic structure for accessibility.
- **CSS3** – Styling, responsive design, and animations.
- **Flexbox & Grid** – Layouts for alignment and positioning.
- **Google Fonts** – `Roboto` font for modern typography.
- **Font Awesome** – Icons for music and UI enhancement.
- **Media Queries** – Ensuring responsiveness across devices.

## Animations & Interactivity
- **Fade In:** Smoothly appear sections while scrolling.
- **Slide Up:** Headings appear sliding from bottom.
- **Floating Effect:** Images (like iPad/iPhone) gently float.
- **Button Hover Effects:** Buttons scale slightly and change opacity on hover.
- **Navigation Hover:** Links get a bottom border on hover.

---

## Responsive Design
- **Small Devices (0–480px):**
  - Reduced font sizes for headings.
  - Background images adjusted.
  - Sections stack vertically.
- **Medium Devices (481px–768px):**
  - Adjusted image widths.
  - Section margins reduced.
  - Flex layouts adjust for tablets.
- **Large Devices (769px–1024px):**
  - Font sizes and images scale appropriately.
  - Flexible gaps and paddings for readability.
